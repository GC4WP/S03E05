# S06E05 

- 以小组为单位，结合本节课讲的“跟圣经学写作”，
- 写一篇任意主题、任意题材文章，
- 要求至少使用圣经里的“微言大义”技巧，
- 字数1000字以内，
- 要求千锤百炼，听觉悦耳，宛若经文。

## 轮值和合升级

- 1000字,分5节, 每节215字;
- 各自认领
    + 不要求有连贯的情节
    + 只要求不能撞车....
- 先写出来, 然后相互念诵, 和合另外4节;
- 这样等于所有段落在5天里都被和合了至少5次
